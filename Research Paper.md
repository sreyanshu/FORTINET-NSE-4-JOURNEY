## points to be noted
- **TITLE** keeping the title relatable and keeping the keywords interesting and prominent.
	- problem to solve
	- method taken to solve it
	- area of working
- **ABSTRACT** 
	- Sort summary of entire paper
	- explaining problem
	- 150-200 words
	- problem -> Objective -> Method ->Results -> Conclusion
- **INTRODUCTION**
	- Background -> Problem -> Research Gap -> Objective -> Contribution
	- Build connection from start till end from problem to solution to process of solving
	- keeping reference point to the research.
	- Solution available but (problem still recites)
	- our solution in short
	- the main contribution of this paper
	- index in one para
